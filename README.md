## 📖 关于我们

**钛极数字** 专注于为中小企业提供 **定制化数字化转型解决方案**。

我们针对不同行业特点，提供标准化+定制化的数字化系统，包含：门店收银、进销存管理、客户CRM、会员营销、线上商城等一体化解决方案。

本站展示了我们为不同行业准备的营销落地页模板，所有模板都是 **响应式设计**，完美适配移动端和桌面端。

## 🚀 快速开始

1. 克隆项目
```bash
git clone https://github.com/yuj1nfeng/yuj1nfeng.github.io.git
cd yuj1nfeng.github.io
```

2. 本地预览
```bash
# 使用 Python 启动简易服务器
python3 -m http.server 8000
# 或者使用 PHP
php -S localhost:8000
```

3. 打开浏览器访问 `http://localhost:8000`

## 📝 自定义修改

1. **样式**：修改对应 `style/*.css` 文件
2. **脚本**：修改对应 `script/*.js` 文件
3. **内容**：直接编辑 HTML 文件修改文字和链接

## 🌐 部署

### GitHub Pages（推荐）

1. Fork 这个仓库
2. 进入仓库 Settings → Pages
3. 选择 `main` 分支保存
4. 访问 `https://<your-username>.github.io/` 即可

### 其他静态托管

直接把所有 HTML/CSS/JS 文件上传到你的静态托管服务即可。
